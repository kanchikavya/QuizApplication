
# Online Quiz Application

## Project Overview
This is an interactive **Online Quiz Application** developed using **React.js**, **JavaScript**, **HTML**, and **CSS**. The application allows users to answer multiple-choice questions in a fun and engaging way.

## Features
- Multiple-choice quiz functionality.
- Timer to limit the time for answering each question.
- Automatic score calculation based on user responses.
- Display of final result and score summary after quiz completion.
- User-friendly interface and responsive design.

## Technologies Used
- **React.js** for frontend development
- **JavaScript** for application logic
- **HTML & CSS** for layout and styling
